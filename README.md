# ColorSimulatorRojo
Generated by [Rojo](https://github.com/rojo-rbx/rojo) 7.6.1.

## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "ColorSimulatorRojo.rbxlx"
```

Next, open `ColorSimulatorRojo.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).

## Weekly update workflow

- Colors and eggs: edit `src/shared/PetData.luau`, then test hatch, inventory, equip best, and merge.
- Worlds/zones: edit `src/shared/ColorData.luau`; zone generation, world UI, and egg gating read from that order.
- Codes: edit `src/shared/CodeData.luau`; server validation lives in `src/server/CodeHandler.server.luau`.
- Economy knobs: edit `src/shared/GameBalance.luau` and upgrade levels in `src/shared/UpgradeData.luau`.

## Update Board workflow

1. Open `src/shared/UpdateLogData.luau`.
2. Change `CurrentVersion`, `LastUpdated`, `Latest.Changes`, `FeaturedCode`, `ComingSoon`, or `PlaytestFocus`.
3. Save and let Rojo sync.
4. Publish or restart the server.
5. The lobby Update Board rebuilds its SurfaceGui from that config when the server starts.
